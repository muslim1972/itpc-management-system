# ITPC Management System

A modern frontend application built with React, Tailwind CSS, and React Router.

## Features

- Login page with responsive design
- Main dashboard page with navigation
- Responsive Navbar component
- React Router for navigation

## Getting Started

### Installation

Install dependencies:

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

Build for production:

```bash
npm run build
```

### Preview

Preview the production build:

```bash
npm run preview
```

## Routes

- `/` - Login page
- `/main` - Main dashboard page

## Tech Stack

- React 18
- React Router DOM
- Tailwind CSS
- Vite
